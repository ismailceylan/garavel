## Installation
1. **Clone the repository:**
	```bash
	git clone https://github.com/yourusername/garavel.git
	```
2. **Navigate to the project directory:**
	```bash
	cd garavel
	```
4. **Install dependencies (if using Composer):**
	```bash
	composer install
	```
